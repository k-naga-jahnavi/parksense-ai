# ParkSense AI

AI-driven parking intelligence platform for analyzing parking violations and visualizing congestion patterns through interactive analytics and geospatial insights.

---

## Overview

ParkSense AI is a full-stack web application built to identify parking hotspots and provide visibility into congestion caused by illegal and unmanaged parking.

The platform processes parking violation data, generates analytical insights, and presents them through an interactive dashboard and map-based visualization.

---

## Features

* Parking hotspot detection
* Vehicle analytics
* Violation analytics
* Junction-based monitoring
* Interactive dashboard
* Bangalore heatmap visualization
* Data-driven congestion insights
